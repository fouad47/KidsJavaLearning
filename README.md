## ğŸ® How to Run Each Chapter

### Step 1: Open Your Favorite Java IDE
- You can use IntelliJ IDEA, Eclipse, or VS Code
- Make sure Java is installed on your computer

### Step 2: Navigate to a Chapter
Each chapter is in its own package like:
```
src/com/kids/java/chapter01_basics/
src/com/kids/java/chapter02_variables/
src/com/kids/java/chapter03_conditions/
... and so on!
```

### Step 3: Run the Main Class
Each chapter has a `Main.java` file. Just run it!

**Example for Chapter 1:**
```bash
cd src
javac com/kids/java/chapter01_basics/*.java
java com.kids.java.chapter01_basics.Main
```

Or simply right-click `Main.java` in your IDE and select "Run"!
